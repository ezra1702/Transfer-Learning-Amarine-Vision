## 1. Apa itu Python?

Python adalah **bahasa pemrograman tingkat tinggi (high-level language)** yang dirancang agar **mudah dibaca, dipelajari, dan digunakan**. Python banyak dipakai untuk:

* Pengembangan perangkat lunak
* Data Science & AI
* Machine Learning
* Otomasi
* Robotika & Computer Vision

---

## 2. Sejarah Python (Kapan Python Dibuat?)

Python pertama kali dibuat oleh **Guido van Rossum** pada **akhir tahun 1980-an** dan dirilis secara resmi pada tahun:

> 📅 **1991**

Tujuan utama Guido membuat Python adalah:

* Membuat bahasa yang **mudah dipahami manusia**
* Mengurangi kerumitan bahasa seperti C dan C++
* Meningkatkan produktivitas programmer

Nama **Python** sendiri diambil dari acara TV Inggris:

> **Monty Python’s Flying Circus**
> (bukan dari ular 🐍)

---

## 3. Python Termasuk Bahasa Apa?

Python termasuk dalam kategori:

### 🔹 Berdasarkan Tingkat Bahasa

* **High-Level Language**

  * Mudah dibaca
  * Mirip bahasa manusia
  * Tidak perlu mengatur memori secara manual

### 🔹 Berdasarkan Cara Eksekusi

* **Interpreted Language**

  * Kode dijalankan baris per baris
  * Lebih mudah debugging
  * Lebih fleksibel

### 🔹 Berdasarkan Paradigma

Python mendukung banyak paradigma:

* Procedural Programming
* Object-Oriented Programming (OOP)
* Functional Programming

## 6. Hubungan Python dengan Divisi Programming Vision pada Robot AUV

### 🌊 Apa itu Robot AUV?

**AUV (Autonomous Underwater Vehicle)** adalah robot bawah air yang dapat bergerak **secara mandiri tanpa kendali langsung manusia**.

Robot AUV biasanya digunakan untuk:

* Eksplorasi bawah laut
* Pencarian objek
* Kompetisi robotika
* Penelitian kelautan

---

## 7. Apa itu Programming Vision?

Programming Vision (Computer Vision) adalah bidang yang membuat **robot dapat “melihat” dan memahami lingkungan** menggunakan kamera atau sensor visual.

---

## 8. Peran Python dalam Vision Robot AUV

Python sangat penting dalam divisi Vision karena:

### 🔹 1. Pengolahan Citra (Image Processing)

Menggunakan library:

* **OpenCV**
* **NumPy**
* **Matplotlib**

Contoh:

* Deteksi warna
* Deteksi bentuk
* Tracking objek di dalam air

---

### 🔹 2. Machine Learning & AI

Python digunakan untuk:

* Klasifikasi objek
* Deteksi target
* Pengenalan pola bawah air

Library yang digunakan:

* TensorFlow
* PyTorch
* Scikit-learn

---

### 🔹 3. Integrasi dengan Sistem Robot AUV

Python bisa terhubung dengan:

* Kamera
* Sensor
* ROS (Robot Operating System)
* Microcontroller (Arduino, STM32)
